# IA liquidaciones agropecuarias

App Streamlit para procesar múltiples PDFs de liquidaciones (primarias/secundarias) y exportar 3 Excels:
- Ventas (modelo HWVta1modelo)
- CPNs (resumen + mercadería entregada)
- Gastos (modelo compras HWCpra1)

## Ejecutar local
```bash
pip install -r requirements.txt
streamlit run app.py
```
